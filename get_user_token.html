Apple Music User Token

Sign in with your Apple ID to get a Music User Token for API access.

Generate your developer token first:

python -c "from playlist_creator.apple_music import AppleMusicClient; c = AppleMusicClient(); print(c._generate_token())"

Or set env vars and run your existing CLI to print it. Paste the JWT above, then click Authorize. The user token is valid for ~6 months.
